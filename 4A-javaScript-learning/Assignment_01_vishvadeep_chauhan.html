Q2: what is the difference between console log, document.write and document.getElementById... with syntax(3 marks)

ans:

  1. console.log() :- it is used for print the variable in the console
    syntax: console.log(variable_name)
  2. document.write() :- it is used to write the html direct to the web pages it is not recommended because it will remove the entire web content
    syntax: document.write("html code")
  3. document.getElementById() is used to get the element by id it will show the output in web page
    syntax: document.getElementById("id_name").innerHTML = "html code"
